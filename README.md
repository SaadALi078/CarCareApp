A modern Android application to manage car maintenance and services, built using:

- 🔥 Firebase Authentication (Login & Signup)
- 🧠 MVVM Architecture
- 🎨 Jetpack Compose UI
- ✅ Modern Clean Code Practices

## Features
- User Login & Signup with Firebase
- MVVM-based architecture
- Jetpack Compose for UI
- Firebase Integration

## Tech Stack
- Kotlin
- Firebase Auth
- Jetpack Compose
- MVVM Architecture
